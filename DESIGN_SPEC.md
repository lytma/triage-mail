# Design Spec — Triage Mail — calm, keyboard-first email triage system

High-volume knowledge workers need a tool that feels like a precision instrument: neutral, scannable, and calm under hundreds of items, with a single confident accent reserved for what matters. The three directions offer (a) a clean minimal light tool, (b) a sharp dark power-tool, and (c) a structured premium editorial feel — all WCAG AA, all keyboard-first. Key screens: Review queue (accent highlights importance, dense rows, inline reply/forward/archive, visible focus ring); Category folder view (bulk-select, flag indicators via semantic color, archive actions); Compose (account selector, clean editor surface, direct labels); Settings — connected accounts & plain-English triage rules (rules rendered verbatim, reconnect banner on lost access); Triage summary stats (calm data viz on neutral cards); Demo account (seeded sample emails mirroring real triage states).
